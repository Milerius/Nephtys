nephtys::client::config
=======================

.. doxygenstruct:: nephtys::client::config
   :project: nephtys
   :members:

Below is an example of a valid configuration for the nephtys client:

.. code-block:: json

    {
       "window":{
          "size":{
             "height":1200,
             "width":800
          },
          "title":"nephtys",
          "fullscreen":false
       }
    }

See also :ref:`nephtys::utils`, :ref:`nephtys::window::win_cfg`.
